### Projects

**[ButtonLab](https://github.com/WebDevAmey/buttonlab)** — `Next.js` `TypeScript` `Tailwind CSS` `Motion` `Radix UI`
A curated, copy-paste component gallery of interactive buttons — hover, press, and steal the code. App Router architecture with per-component live preview pages (`/button/<id>`), an interactive playground for prop tweaking, and multi-format code export (JSX/CSS/Tailwind). Built with Motion for hover/tap micro-interactions and Radix UI primitives for accessible playground controls. Contributor-friendly by design — new buttons ship as a single file plus one registry entry.
