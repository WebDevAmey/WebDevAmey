### 📦 Projects

**[ButtonLab](https://github.com/WebDevAmey/ButtonLab)** — `Next.js` `TypeScript` `Tailwind CSS` `Motion` `Radix UI`
Curated, copy-paste component gallery of interactive buttons — hover, press, and steal the code. Per-component live preview pages (`/button/<id>`), an interactive prop playground, and multi-format code export (JSX/CSS/Tailwind). Motion-driven micro-interactions, Radix UI-powered accessible controls.

**[HoveraUI](https://github.com/WebDevAmey/HoveraUI)** — `TypeScript`
A UI component library exploring hover-driven interaction patterns — reusable, styled components built for drop-in use in modern React/Next.js apps.

**[TechPulse](https://github.com/WebDevAmey/TechPulse)** — `TypeScript`
Tracks tech product launches and the discourse around them — aggregating signal from launch platforms and social discussion into one feed.

**[ai-summit-agent](https://github.com/WebDevAmey/ai-summit-agent)** — `TypeScript` · [Live](https://ai-summit-agent.vercel.app)
An agent/aggregator that surfaces everything related to the AI Impact Summit 2026 — talks, news, and discussion in one place.

**[Karagiri](https://github.com/WebDevAmey/FSDL_MiniProject)** — `JavaScript` · [Live](https://karagiri-azure.vercel.app)
A marketplace platform connecting local artisans with buyers — product listings, storefronts, and discovery for small-scale makers.

**[Lead-Agent](https://github.com/WebDevAmey/Lead-Agent)** — `HTML/JS`
An agent that automates lead generation for businesses — sourcing and surfacing prospects without manual research.

**[Simple RAG Chatbot](https://github.com/WebDevAmey/Simple-Rag-Chatbot-)** — `TypeScript`
A retrieval-augmented generation chatbot with a full frontend — grounding LLM responses in a custom knowledge base via a RAG pipeline.
